## Setup Instructions
### Prerequisites
- A modern web browser.
- An active internet connection.

### Installation
1. Clone the repository:  
   `git clone https://github.com/peachyecho/vigilant-octo-tribble.git`
2. Navigate to the project directory:  
   `cd vigilant-octo-tribble`
3. Install dependencies:  
   `npm install` 
4. Start the application:  
   `npm start`

### Configuration
- Update configuration settings in the `config.js` file according to your needs.
